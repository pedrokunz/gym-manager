# Gym Manager Frontend

Vue 3 frontend for Gym Manager.

## Features

- Built with Vite & Vue 3
- Vue Router for navigation
- Modern UI with standard CSS

## Modules

- **Member Management**: Complete CRUD operations for gym members.
- **Membership Plans**: Viewing and discount calculation engine.
- **Gym Classes**: GraphQL-powered class viewer.
- **Sidebar**: Context-aware sidebar with recent plan data.

## Testing

- **Vitest**: Unit testing for pricing and discount logic.
- **Playwright**: End-to-end testing for core user flows (Members, Plans).

## Quick Start


### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```


### Build for production

```bash
npm run build
```
